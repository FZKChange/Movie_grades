# Movie_grades
  一个有关电影评分回归的集合，包括情感分析模型和打分回归算法，一共六个模型代码，两个回归算法结合三个语言分析模型。
  newdata.csv是爬取豆瓣上电影的中文短评和对应打分，new_test.csv用的是张艺谋的《无极》，newdata.csv一共是39部电影，3765条评论，1星到5星近似均衡分布，需要提前下载好torch等对应库。
